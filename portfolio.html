HW 1

In this homework I learnt how to implement the "scout-rule". The importance of refactoring an existing code and making it into something more efficient and concise is what Web-Developers are expected to do. This forced me to research a few topics and be proficient in the art of "Google-Fu".

HW 2

In this homework I learnt how to implement the learnings of HTML, CSS and Bootstrap to create my first ever Portfolio. For a Web-Developer his/her Portfolio is the most important arrow in their quiver. It shows potential employers what I have achieved so far. Although this is my first Portfolio, there's only one way to go from here and that's up!

HW 3

In this homework we we were asked to create a random password generator using numbers, alphabets and special characters. The user gets to decide the length of the password between 8 and 128 characters and what type of characters do they want. Based on their selection the password generator gives the user a randomized password.

HW 4

In this homework we were asked to use JS to create a basic quiz that asks the user a few multiple choice questions and get and store their answer in local storage. If the user selected a wrong choice they will be penalized by deduction of 5 seconds. Also make use of local storage to keep tab of multiple user's high scores.

HW 5

In this homework we were asked to make a daily planner using moment.js library. The daily planner would have color coded time slots that worked in accordance with moment.js library. The tasks when entered should stay persistent even if the user hits reload until cleared manually.
